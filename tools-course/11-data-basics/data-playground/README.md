# Data Playground

This project demonstrates the 4 core data formats used in engineering.

## 1. JSON (samples.json)
* **Use:** Machine-to-machine data exchange (APIs).
* **Structure:** Strict rules, curly braces, double quotes.

## 2. YAML (config.yaml)
* **Use:** Human-readable configuration (Docker, Kubernetes).
* **Structure:** Indentation-based, cleaner to read.

## 3. CSV (samples.csv)
* **Use:** Tabular data, spreadsheet exports.
* **Structure:** Rows and columns separated by commas.

## 4. ENV (.env.example)
* **Use:** Secrets and environment variables.
* **Rule:** Never commit the real .env file.
